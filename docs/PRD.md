# Swanipe — Product Requirements Document

> **Emotional Hook**: Free RPCs are 1-2 seconds behind. Traders miss launches.

## Problem Statement
Default Solana RPCs have high latency. No benchmark comparison exists in-app.

## Solution Overview
Use RPC Fast as primary infra. Cross-submit for credits. Integration = swap 1 URL.

## Core Features (MVP)
1. RPC Fast as primary infrastructure
2. Latency comparison chart
3. WebSocket subscriptions

## Success Metrics
- All SDK features demonstrably integrated
- Demo flow works end-to-end

## Out of Scope
- ❌ Custom RPC logic
- ❌ Failover
- ❌ Multi-region
